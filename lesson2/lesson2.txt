Попробуем поработать с командами автоматизации для SNVNA,
для этого ознакомьтесь с документом Doc\Руководство программиста\SVNA_руководство_по_программированию.pdf
Вас интересует всё что связанно с SCPI.

Задания:

1. Запустите программу SocketTesterPrjScript.exe и установите соединение с программой SNVNA. В программе SNVNA должен быть выбран 16-ти портовый демо прибор.
Настройки подключения можно посмотреть в боковом меню в разделе settings.

2.После установки соеденения отправте запрос *IDN? , в ответ вы должны получить что-то похожее на 

<- COPPER MOUNTAIN TECHNOLOGIES, SN5090-16, 123456, 22.6.0-beta1/2


Информацию по командам вы можете найти в Doc\Руководство программиста\SVNA_руководство_по_программированию.pdf
3. Попробуйте с помощью комманд автоматизации сделать следующие действия:
	a. Добавить трассу в канал.
	b. Удалить трассу в канале.
	с. Изменить параметр трассы S11 на S22
	d. Добавить маркер на трассу
	e. Изменить частоту маркера по оси стимула(X) на 1Ghz
	f. Удалить маркер
	g. Добавить два маркера, один из них установить на частоту 3Ghz другой на частоту 3.5Ghz(это несколько команд, нужно выполнить подряд)

Ответ на 3 пункт запишите в отдельный файл .txt в папке lesson2 по типу

a. Добавить трассу в канал -> "комманда"
...
g. g. Добавить два маркера, один из них установить на частоту 3Ghz другой на частоту 3.5Ghz(это несколько команд, нужно выполнить подряд) -> "комманда"

Сделайте коммит, и запушьте изменения в репозиторий.
